# data/

`orogenic_gold_ages.csv` — orogenic-gold deposit ages (deposit, age_ma, age_err,
method, craton) re-analysed in the note.

**Source:** the author's own four-province metallogeny dataset,
Chen (2026), *Charge-and-release rhythms in ore-forming systems*, Zenodo
https://doi.org/10.5281/zenodo.20768849 (file `data/orogenic_gold.csv`). Compiled
there from published U-Pb xenotime/monazite, Re-Os and Ar-Ar geochronology.
Included here so the re-audit is self-contained.
